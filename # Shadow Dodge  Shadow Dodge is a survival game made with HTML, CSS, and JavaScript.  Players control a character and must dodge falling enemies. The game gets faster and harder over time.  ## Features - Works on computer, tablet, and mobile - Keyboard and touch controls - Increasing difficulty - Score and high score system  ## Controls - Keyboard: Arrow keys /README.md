# Shadow Dodge

Shadow Dodge is a survival game made with HTML, CSS, and JavaScript.

Players control a character and must dodge falling enemies.
The game gets faster and harder over time.

## Features
- Works on computer, tablet, and mobile
- Keyboard and touch controls
- Increasing difficulty
- Score and high score system

## Controls
- Keyboard: Arrow keys / A D
- Mobile: On-screen buttons

## Goal
Survive as long as possible.
